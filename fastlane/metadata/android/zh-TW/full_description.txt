Barcode Scanner 是一個免費與開放原始碼應用程式，允許讀取與產生條碼。它可以收集關於食品、化妝品與書籍的資訊，

以此應用管理不同的條碼格式：
• 二維條碼：QR Code, Data Matrix, PDF 417, AZTEC
• 一維條碼：EAN 13, EAN 8, UPC A, UPC E, Code 128, Code 93, Code 39, Codabar, ITF

在掃描期間收集關於產品的資訊：
• 食品資訊收集來自 Open Food Facts
• 化妝品資訊收集來自 Open Beauty Facts
• 寵物食品資訊收集來自 Open Pet Food Facts
• 書籍資訊收集來自 Open Library
• 音樂 CD、黑膠唱片資訊收集來自 MusicBrainz

應用程式功能：
• 只需將您的智慧型手機對準一個條碼，或者也可以通過掃描一個圖片，則即可立即獲得關於它的資訊。
• 通過簡單的掃描，讀取名片、加入新聯絡人、加入新事件至您的日程、開啟 URL 甚至連接 Wi-Fi。
• 掃描食品條碼以獲得關於它們的成分資訊，感謝 Open Food Facts 與 Open Beauty Facts 資料庫。
• 搜尋關於您掃描的產品資訊，在不同的網站上快速搜尋，例如 Amazon 或 Fnac。
• 通過歷史紀錄工具，持續追蹤所有您已掃描的條碼。
• 產生您自己的條碼。
• 以不同的色彩自訂介面，使用淺色主題或深色主題。此應用整合 Android 12 的功能，允許根據您的桌布調整色彩。
• 完整翻譯為英語、西班牙語、法語、德語、俄語以及中文。

此應用程式尊重您的隱私。它不含有任何追蹤器，也不收集任何資料。
